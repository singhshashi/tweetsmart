Who made this and why?

Hi, I am Shashi (@_singhshashi) and I have programmed mostly on the Windows stack and wanted to get my hands dirty building something on Linux. Amongst the many other ideas that I had, this seemed like something which people would find useful. At the same time, it did not seem like a very big project and was also an opportunity to learn Reactjs, something which I wanted to do.

The client side app is build using Flux, Reactjs and the server side api is built using expressjs. The source code of this project is available on Github.
