Who made this and why?

TweetSmart was conceptualized by Shashi when he came across tweet storms by JP_LokSatta. The tweet storms by JP_LokSatta were always numbered perfectly. This was something which could not be done without some effort. Thus, this seemed like a perfect use case to build something, when learning new technologies, that would be used by people.

The client side app and Chrome extension is built using Flux, Reactjs and the server side api is built using expressjs. The source code of part of this project is available on Github.
